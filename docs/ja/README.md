# Synuraへようこそ！

## Synuraとは？
Synuraは、「拡張機能（Extensions）」と呼ばれる強力なミニアプリを使用して、さまざまなソースからのコンテンツを閲覧できる多目的なアプリケーションです。ブラウザのようなものと考えてください。ただし、ウェブサイトの代わりに拡張機能を使用して、クリーンなネイティブアプリ形式でコンテンツを取得します。

## ユーザーのためのコアコンセプト

*   **拡張機能の検出**: ドメイン（例：`example.com`または`https://example.com`）を入力すると、そのドメインから`synura.js`ファイルを自動的に取得します。プロトコルが指定されていない場合、デフォルトで`https://`が使用されます。これは、公式サイトから拡張機能をインストールする主な方法です。
*   **直接インストール**: 完全なURL（例：`https://raw.githubusercontent.com/user/repo/main/synura.js`）を入力して、特定の拡張機能スクリプトをインストールします。**セキュリティ上の注意**: この方法は、悪意のあるコードの実行を防ぐため、信頼できるドメイン（GitHub、GitLabなど）に制限されています。一般的なドメインには使用しないでください。
*   **ホワイトリスト検証**: 直接URLインストールは、セキュリティのために許可されたドメインのホワイトリストに対して検証されます。ドメイン検出は、探索を許可するためにこのチェックをバイパスします。
*   **拡張機能**: これらはコンテンツを取得して表示する小さなプラグインです。たとえば、ニュースサイト、動画プラットフォーム、またはソーシャルメディアフィード用の拡張機能を持つことができます。新しい拡張機能をインストールして、Synuraでできることを拡張できます。
*   **ランタイム（Runtimes）**: 拡張機能を開くと、「ランタイム」で実行されます。ウェブブラウザで複数のタブを開くのと同じように、一度に複数のランタイムを開くことができます。各ランタイムは、拡張機能の個別のインスタンスです。それらを切り替えたり、同じ拡張機能に対して複数のランタイムを持ったりすることもできます。
*   **ブックマーク**: 面白いものを見つけましたか？現在のビューをブックマークして、後で保存することができます。ブックマークはビューの正確な状態を保存するため、いつでもそこに戻ることができます。

## アプリ内の操作

### メイン画面
アプリのメイン画面は、ランタイムを管理する場所です。上部バー（アプリバー）が主なナビゲーションツールです。

### アプリバー

アプリバーには、コンテンツのナビゲートと管理に役立ついくつかのアイコンがあります。一部のアイコンには、長押しでアクセスできる**隠しショートカット**があります。

*   **`+`（追加）**:
    *   **タップ**: 新しいランタイムを開きます。インストール済みの拡張機能を選択するか、ウェブサイトのドメイン/URLを入力して新しいものをインストールできます。
    *   **長押し**: **拡張機能管理**画面を開き、インストール済みの拡張機能に関する詳細を表示します。
*   **ドロップダウンメニュー（中央）**: 現在アクティブなランタイムを表示します。タップして開いているランタイムを切り替えるか、ドロップダウンを左右にスワイプしてそれらを循環させます。
*   **`X`（閉じる）**: 現在のランタイムを閉じます。
*   **`↻`（更新）**: *開発者モードでのみ表示されます。* 現在の拡張機能をソースから更新します。
*   **`✨`（AI）**:
    *   **タップ**: 素早いアクション（要約、翻訳など）のために**AIメニュー**を開きます。
    *   **長押し**: **AI設定**を開き、プロバイダーと設定を構成します。
*   **`☆`（ブックマーク追加）**:
    *   **タップ**: 現在のビューをブックマークに保存します。
    *   **長押し**: **ブックマーク**リストに直接移動します。
*   **`🔖`（ブックマーク）**: 保存されたブックマークのリストを表示します。
*   **`⚙️`（設定）**: メイン設定画面を開きます。

画面が狭すぎる場合、一部のオプションは3点メニューに移動することがあります。

### AIボタン（`✨`）
アプリバーの**AIボタン**をタップして、**AIメニューダイアログ**を開きます。これにより、現在のビューに対してオンデマンドのAI機能が提供されます。

*   **要約**: 画面上のコンテンツのAI生成による要約を素早く取得します。
*   **翻訳**: コンテンツをターゲット言語（AI設定で構成）に翻訳します。
*   **カスタムプロンプト**: AIにコンテンツを分析させるための独自の指示を入力します。
*   **外部AIに共有**: 現在のビューのコンテンツを、デバイス上のChatGPTやGeminiなどの外部AIアプリにエクスポートします。
*   **キャッシュ切り替え**: キャッシュされたAI結果を使用するか、新しい分析を強制するかを制御します。

詳細なAI構成については、**設定 > AI設定**に移動してください。ここでは次のことができます。
*   お好みのAIプロバイダー（Gemini、 OpenAI、 DeepSeek、 Claude）を構成します。
*   翻訳のソース言語とターゲット言語を設定します。
*   分析検索範囲（リストビューのみ詳細あり）とプロファイル（要約、説明、簡素化、ファクトチェック、批評、洞察）を選択します。
*   要約の長さの設定を調整します。
*   トークン使用統計を表示します。
*   各プロバイダーのAPIキーを管理します。

### ブックマーク
ブックマーク画面には、保存されたすべてのビューが表示されます。

*   **ビュースナップショット**: ブックマークをタップすると、保存したときのページの**キャッシュされたスナップショット**が開きます。これは、インターネット接続なしで情報をすばやく参照するのに最適です。
*   **ビューの復元**: ページと再度対話する（リンクをクリックする、データを更新するなど）には、**復元アイコン**を探してください。これをタップすると、拡張機能に再接続し、新しいランタイムでビューを復活させます。

## 設定（`⚙️`）

設定画面では、Synuraエクスペリエンスのほぼすべての側面を微調整できます。

### 拡張機能
*   **新しい拡張機能のインストール**: アプリバーの**`+`**ボタンをタップし、ウェブサイトのドメイン（例：`https://example.com`）を入力します。サイトがSynuraをサポートしている場合、拡張機能は自動的に検出され、インストールされます。
*   **拡張機能の管理**: **管理**をタップしてインストール済みの拡張機能のリストを表示し、更新または削除できます。

### 外観
*   **コンテンツ密度の調整**: スライダーを使用して、コンテンツをより広げたり、よりコンパクトにしたりします。リストやカードへの影響をライブプレビューで確認できます。
*   **カラーテーマ**: **ライト**、**ダーク**、**Monokai**の配色から選択して、アプリの外観をカスタマイズします。
*   **フォントの太さ**: テキストの太さを好みに合わせて調整します（例：細字、標準、太字）。
*   **言語**: アプリケーションの言語を設定します。特定の言語を選択するか、システムのデフォルトに従わせることができます。

### 動作
*   **ネットワークタイムアウト**: アプリがネットワークリクエストからの応答を待機する時間を1〜60秒の間で設定します。
*   **プロキシ設定**: ネットワークリクエスト用のプロキシサーバーを構成します。
*   **キャッシュ設定**: スペースを解放するためにキャッシュされたデータをクリアするなど、アプリケーションのキャッシュを管理します。
*   **GIFアニメーション**: アニメーションGIFの再生方法を制御します：**オフ**（静止画）、**1回**（1回再生）、または**ループ**（連続再生）。

### 動画と音声
*   **動画自動再生**: 画面に表示されたときに動画の再生を自動的に開始するかどうかを制御するスイッチ。
*   **動画バックグラウンド再生**: これを有効にすると、別の場所に移動したり別のアプリに切り替えたりした後でも、動画の音声を聞き続けることができます。
*   **他とミックス**: Synuraの音声を他のアプリの音声と同時に再生できるようにします。
*   **ライブ再生DVR時間**: ライブストリームの場合、過去に遡ってシークするために放送を何時間保持するかを選択します（0〜6時間）。

### プライバシーとセキュリティ
*   **設定の管理**: 保存および共有されるデータを制御するためのさまざまなプライバシーおよびセキュリティオプションを構成します。

### 情報
*   **オープンソースライセンス**: Synuraの動作を支援するオープンソースソフトウェアのライセンスを表示します。
