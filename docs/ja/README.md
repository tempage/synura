# Synuraへようこそ！

## Synuraとは？
Synuraは、「拡張機能（Extensions）」と呼ばれる強力なミニアプリを使用して、さまざまなソースからのコンテンツを閲覧できる多用途なアプリケーションです。ブラウザのようなものと考えてください。ただし、Webサイトの代わりに拡張機能を使用して、クリーンなネイティブアプリ形式でコンテンツを取得します。

## ユーザー向けの主要な概念

*   **拡張機能の検出**: ドメイン（例：`example.com`または`https://example.com`）を入力すると、そのドメインから`synura.js`ファイルを自動的に取得します。プロトコルが指定されていない場合は、デフォルトで`https://`が使用されます。これは、公式サイトから拡張機能をインストールする主な方法です。
*   **直接インストール**: 特定の拡張機能スクリプトをインストールするには、完全なURL（例：`https://raw.githubusercontent.com/user/repo/main/synura.js`）を入力します。**セキュリティ上の注意**: この方法は、悪意のあるコードの実行を防ぐために、信頼できるドメイン（GitHub、GitLabなど）に制限されています。一般的なドメインにはこれを使用しないでください。
*   **ホワイトリスト検証**: 直接URLインストールは、セキュリティのために許可されたドメインのホワイトリストに対して検証されます。ドメイン検出は、探索を許可するためにこのチェックをバイパスします。
*   **拡張機能**: これらは、コンテンツを取得して表示する小さなプラグインです。たとえば、ニュースサイト、動画プラットフォーム、またはソーシャルメディアフィード用の拡張機能を持つことができます。新しい拡張機能をインストールして、Synuraでできることを拡張できます。
*   **ランタイム（Runtimes）**: 拡張機能を開くと、それは「ランタイム」で実行されます。Webブラウザに複数のタブがあるのと同じように、複数のランタイムを同時に開くことができます。各ランタイムは、拡張機能の個別のインスタンスです。それらを切り替えることができ、同じ拡張機能に対して複数のランタイムを持つこともできます。
*   **ブックマーク**: 興味深いものを見つけましたか？現在のビューをブックマークして、後で保存できます。ブックマークはビューの正確な状態を保存するため、いつでもそこに戻ることができます。

## アプリ内の移動

### メイン画面
アプリのメイン画面は、ランタイムを管理する場所です。上部のバー（アプリバー）が主なナビゲーションツールです。

### アプリバー

アプリバーにはいくつかのアイコンがあります：

*   **`+` (追加)**: これをタップして、新しいランタイムを開きます。インストールされている拡張機能を選択するか、Webサイトのドメインを入力して新しい拡張機能をインストールできます。
*   **ドロップダウンメニュー (中央)**: 現在アクティブなランタイムを表示します。タップすると、開いているすべてのランタイムのリストが表示され、切り替えることができます。ドロップダウンを左右にスワイプしてすばやく切り替えることもできます。
*   **`X` (閉じる)**: 現在のランタイムを閉じます。
*   **`☆` (ブックマークに追加)**: これをタップして、現在のビューをブックマークに保存します。
*   **`🔖` (ブックマーク)**: 保存されたブックマークのリストに移動します。
*   **`⚙️` (設定)**: 設定画面を開き、Synuraをカスタマイズできます。

画面が狭すぎる場合、これらのオプションは右側の3点メニューに折りたたまれます。

### ブックマーク
ブックマーク画面には、保存されたすべてのビューが表示されます。

*   **ビュースナップショット**: ブックマークをタップすると、保存したときのページの**キャッシュされたスナップショット**が開きます。これは、インターネット接続なしで情報をすばやく参照するのに最適です。
*   **ビューの復元**: ページを再度操作する（リンクをクリックする、データを更新するなど）には、**復元アイコン**を探します。これをタップすると、拡張機能に再接続し、新しいランタイムでビューを復活させます。

## 設定 (`⚙️`)

設定画面では、Synuraエクスペリエンスのほぼすべての側面を微調整できます。

### 拡張機能
*   **新しい拡張機能のインストール**: アプリバーの**`+`**ボタンをタップし、Webサイトのドメイン（例：`https://example.com`）を入力します。サイトがSynuraをサポートしている場合、拡張機能は自動的に検出され、インストールされます。
*   **拡張機能の管理**: **管理**をタップしてインストールされている拡張機能のリストを表示し、更新または削除できます。

### 外観
*   **コンテンツ密度の調整**: スライダーを使用して、コンテンツをより広く、またはよりコンパクトに表示します。リストやカードへの影響のライブプレビューが表示されます。
*   **カラーテーマ**: **ライト**、**ダーク**、**Monokai**の配色から選択して、アプリの外観をカスタマイズします。
*   **フォントの太さ**: 好みに合わせてテキストの太さを調整します（例：ライト、レギュラー、ボールド）。
*   **言語**: アプリケーションの言語を設定します。特定の言語を選択するか、システムのデフォルトに従わせることができます。

### 動作
*   **ネットワークタイムアウト**: アプリがネットワーク要求からの応答を待機する時間を1〜60秒の間で設定します。
*   **プロキシ設定**: ネットワーク要求用のプロキシサーバーを構成します。
*   **キャッシュ設定**: スペースを解放するためにキャッシュされたデータをクリアするなど、アプリケーションのキャッシュを管理します。
*   **GIFアニメーション**: アニメーションGIFの再生方法を制御します：**オフ**（静止画）、**1回**（1回再生）、または**ループ**（連続再生）。

### 動画と音声
*   **動画の自動再生**: 画面に表示されたときに動画の再生を自動的に開始するかどうかを制御するスイッチ。
*   **動画のバックグラウンド再生**: これを有効にすると、別の場所に移動したり別のアプリに切り替えたりした後でも、動画の音声を聞き続けることができます。
*   **他とミックス**: Synuraからの音声を他のアプリからの音声と同時に再生することを許可します。
*   **ライブ再生DVR時間**: ライブストリームの場合、ブロードキャストの何時間をさかのぼって利用できるようにするかを選択します（0〜6時間）。

### プライバシーとセキュリティ
*   **設定の管理**: さまざまなプライバシーとセキュリティオプションを構成して、保存および共有されるデータを制御します。

### 情報
*   **オープンソースライセンス**: Synuraを動かすのに役立つオープンソースソフトウェアのライセンスを表示します。
