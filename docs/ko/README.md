# Synura에 오신 것을 환영합니다!

## Synura란 무엇인가요?
Synura는 "확장 프로그램(Extension)"이라는 강력한 미니 앱을 사용하여 다양한 소스의 콘텐츠를 탐색할 수 있는 다목적 애플리케이션입니다. 브라우저라고 생각하면 되지만, 웹사이트 대신 확장 프로그램을 사용하여 깔끔한 네이티브 앱 형식으로 콘텐츠를 가져옵니다.

## 사용자를 위한 핵심 개념

*   **확장 프로그램 검색**: 도메인(예: `example.com` 또는 `https://example.com`)을 입력하여 해당 도메인에서 `synura.js` 파일을 자동으로 가져옵니다. 프로토콜이 제공되지 않으면 기본적으로 `https://`가 사용됩니다. 이것은 공식 웹사이트에서 확장 프로그램을 설치하는 기본적인 방법입니다.
*   **직접 설치**: 전체 URL(예: `https://raw.githubusercontent.com/user/repo/main/synura.js`)을 입력하여 특정 확장 프로그램 스크립트를 설치합니다. **보안 참고**: 이 방법은 악성 코드 실행을 방지하기 위해 신뢰할 수 있는 도메인(GitHub, GitLab 등)으로 제한됩니다. 일반 도메인에는 사용하지 마십시오.
*   **화이트리스트 검증**: 직접 URL 설치는 보안을 위해 허용된 도메인 화이트리스트에 대해 검증됩니다. 도메인 검색은 탐색을 허용하기 위해 이 확인을 우회합니다.
*   **확장 프로그램**: 콘텐츠를 가져와 표시하는 작은 플러그인입니다. 예를 들어 뉴스 사이트, 비디오 플랫폼 또는 소셜 미디어 피드를 위한 확장 프로그램을 가질 수 있습니다. 새로운 확장 프로그램을 설치하여 Synura로 할 수 있는 작업을 확장할 수 있습니다.
*   **런타임(Runtimes)**: 확장 프로그램을 열면 "런타임"에서 실행됩니다. 웹 브라우저에 여러 탭이 있는 것처럼 한 번에 여러 런타임을 열어 둘 수 있습니다. 각 런타임은 확장 프로그램의 별도 인스턴스입니다. 이들 간에 전환할 수 있으며 동일한 확장 프로그램에 대해 여러 런타임을 가질 수도 있습니다.
*   **북마크**: 흥미로운 것을 발견했나요? 현재 보기를 북마크하여 나중에 저장할 수 있습니다. 북마크는 보기의 정확한 상태를 저장하므로 언제든지 다시 돌아올 수 있습니다.

## 앱 둘러보기

### 메인 화면
앱의 메인 화면은 런타임을 관리하는 곳입니다. 상단 표시줄(앱 바)은 주요 탐색 도구입니다.

### 앱 바

앱 바에는 여러 아이콘이 있습니다:

*   **`+` (추가)**: 새 런타임을 열려면 이것을 탭하세요. 설치된 확장 프로그램을 선택하거나 웹사이트 도메인을 입력하여 새 확장 프로그램을 설치할 수 있습니다.
*   **드롭다운 메뉴 (중앙)**: 현재 활성 상태인 런타임을 표시합니다. 열려 있는 모든 런타임 목록을 보고 전환하려면 탭하세요. 드롭다운에서 왼쪽이나 오른쪽으로 스와이프하여 빠르게 전환할 수도 있습니다.
*   **`X` (닫기)**: 현재 런타임을 닫습니다.
*   **`☆` (북마크 추가)**: 현재 보기를 북마크에 저장하려면 이것을 탭하세요.
*   **`🔖` (북마크)**: 저장된 북마크 목록으로 이동합니다.
*   **`⚙️` (설정)**: Synura를 사용자 정의할 수 있는 설정 화면을 엽니다.

화면이 너무 좁으면 이 옵션들은 오른쪽의 점 3개 메뉴로 축소됩니다.

### 북마크
북마크 화면에는 저장된 모든 보기가 표시됩니다.

*   **보기 스냅샷**: 북마크를 탭하면 저장했을 때와 동일한 페이지의 **캐시된 스냅샷**이 열립니다. 인터넷 연결 없이 정보를 빠르게 참조하는 데 좋습니다.
*   **보기 복원**: 페이지와 다시 상호 작용하려면(예: 링크 클릭, 데이터 새로 고침) **복원 아이콘**을 찾으세요. 이것을 탭하면 확장 프로그램에 다시 연결하고 새 런타임에서 보기를 다시 활성화합니다.

## 설정 (`⚙️`)

설정 화면에서는 Synura 경험의 거의 모든 측면을 미세 조정할 수 있습니다.

### 확장 프로그램
*   **새 확장 프로그램 설치**: 앱 바의 **`+`** 버튼을 탭하고 웹사이트 도메인(예: `https://example.com`)을 입력하세요. 사이트가 Synura를 지원하는 경우 확장 프로그램이 자동으로 검색되어 설치됩니다.
*   **확장 프로그램 관리**: **관리**를 탭하여 설치된 확장 프로그램 목록을 확인하고 업데이트하거나 제거할 수 있습니다.

### 모양
*   **콘텐츠 밀도 조정**: 슬라이더를 사용하여 콘텐츠를 더 넓게 또는 더 촘촘하게 보이게 만드세요. 목록과 카드에 어떤 영향을 미치는지 실시간 미리보기를 볼 수 있습니다.
*   **색상 테마**: **밝게**, **어둡게**, **Monokai** 색상 구성표 중에서 선택하여 앱의 모양을 개인화하세요.
*   **글꼴 두께**: 텍스트 굵기를 원하는 대로 조정하세요(예: 가볍게, 보통, 굵게).
*   **언어**: 애플리케이션 언어를 설정합니다. 특정 언어를 선택하거나 시스템 기본값을 따르도록 할 수 있습니다.

### 동작
*   **네트워크 시간 초과**: 앱이 네트워크 요청의 응답을 기다리는 시간을 1초에서 60초 사이로 설정하세요.
*   **프록시 설정**: 네트워크 요청을 위한 프록시 서버를 구성합니다.
*   **캐시 설정**: 공간을 확보하기 위해 캐시된 데이터를 지우는 것을 포함하여 애플리케이션의 캐시를 관리합니다.
*   **GIF 애니메이션**: 애니메이션 GIF 재생 방식을 제어합니다: **끄기** (정지 이미지), **한 번** (한 번 재생), 또는 **반복** (계속 재생).

### 비디오 및 오디오
*   **비디오 자동 재생**: 화면에 나타날 때 비디오가 자동으로 재생되는지 여부를 제어하는 스위치입니다.
*   **비디오 백그라운드 재생**: 다른 곳으로 이동하거나 다른 앱으로 전환한 후에도 비디오의 오디오를 계속 들으려면 이 기능을 활성화하세요.
*   **다른 앱과 믹스**: Synura의 오디오가 다른 앱의 오디오와 동시에 재생되도록 허용합니다.
*   **라이브 재생 DVR 시간**: 라이브 스트림의 경우 방송의 몇 시간을 되감기 위해 유지할지 선택하세요(0에서 6시간).

### 개인정보 및 보안
*   **설정 관리**: 저장 및 공유되는 데이터를 제어하기 위해 다양한 개인정보 및 보안 옵션을 구성하세요.

### 정보
*   **오픈 소스 라이선스**: Synura를 구동하는 데 도움이 되는 오픈 소스 소프트웨어의 라이선스를 확인하세요.
