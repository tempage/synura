# 歡迎使用 Synura！

## 什麼是 Synura？
Synura 是一款多功能應用程式，允許您使用稱為「擴充功能」（extensions）的強大迷你應用程式瀏覽來自各種來源的內容。可以把它想像成一個瀏覽器，但您使用的是擴充功能來獲取乾淨、原生應用格式的內容，而不是網站。

## 使用者核心概念

*   **擴充功能發現**：輸入一個網域（例如 `example.com` 或 `https://example.com`）以自動從該網域獲取 `synura.js` 檔案。如果未提供協定，則預設使用 `https://`。這是從官方網站安裝擴充功能的主要方式。
*   **直接安裝**：輸入完整的 URL（例如 `https://raw.githubusercontent.com/user/repo/main/synura.js`）以安裝特定的擴充功能指令碼。**安全提示**：此方法僅限於受信任的網域（如 GitHub、GitLab 等），以防止惡意程式碼執行。請勿對通用網域使用此方法。
*   **白名單驗證**：為了安全起見，直接 URL 安裝將根據允許網域的白名單進行驗證。網域發現繞過此檢查以允許探索。
*   **擴充功能**：這些是獲取和顯示內容的小型外掛程式。例如，您可以擁有一個新聞網站、影片平台或社群媒體來源的擴充功能。您可以安裝新的擴充功能來擴展您可以使用 Synura 做的事情。
*   **執行時 (Runtimes)**：當您打開一個擴充功能時，它在一個「執行時」中執行。您可以同時打開多個執行時，就像在 Web 瀏覽器中有多個分頁一樣。每個執行時都是擴充功能的一個單獨實例。您可以在它們之間切換，甚至可以為同一個擴充功能擁有多個執行時。
*   **書籤**：發現了有趣的東西？您可以將目前檢視新增為書籤以供稍後儲存。書籤儲存檢視的確切狀態，因此您可以隨時返回。

## 在應用中導航

### 主畫面
應用的主畫面是您管理執行時的地方。頂部欄（應用欄）是您的主要導航工具。

### 應用欄

應用欄有幾個圖示：

*   **`+` (新增)**：點選此按鈕打開一個新的執行時。您可以選擇已安裝的擴充功能或輸入網站網域來安裝新的擴充功能。
*   **下拉式選單 (中間)**：顯示目前活動的執行時。點選它可檢視所有打開的執行時列表並在它們之間切換。您也可以在下拉式選單上向左或向右滑動以快速切換。
*   **`X` (關閉)**：關閉目前執行時。
*   **`☆` (新增書籤)**：點選此按鈕將目前檢視儲存到您的書籤。
*   **`🔖` (書籤)**：帶您進入已儲存的書籤列表。
*   **`⚙️` (設定)**：打開設定畫面，您可以在其中自訂 Synura。

如果螢幕太窄，這些選項將摺疊到右側的三點選單中。

### 書籤
書籤畫面顯示您所有儲存的檢視。

*   **檢視快照**：點選書籤會打開頁面在您儲存時的**快取快照**。這非常適合在無需網際網路連線的情況下快速參考資訊。
*   **恢復檢視**：要再次與頁面互動（例如，點選連結、重新整理資料），請查詢**恢復圖示**。點選此圖示將重新連線到擴充功能並在新的執行時中使檢視恢復活動。

## 設定 (`⚙️`)

設定畫面允許您微調 Synura 體驗的幾乎每個方面。

### 擴充功能
*   **安裝新擴充功能**：點選應用欄中的 **`+`** 按鈕並輸入網站網域（例如 `https://example.com`）。如果網站支援 Synura，將自動發現並安裝擴充功能。
*   **管理擴充功能**：點選 **管理** 檢視已安裝的擴充功能列表，您可以在其中更新或刪除它們。

### 外觀
*   **調整內容密度**：使用滑桿使內容看起來更分散或更緊湊。您將看到它如何影響列表和卡片的即時預覽。
*   **顏色主題**：透過在 **淺色**、**深色** 和 **Monokai** 配色方案之間進行選擇來個人化應用的外觀。
*   **字型粗細**：根據您的喜好調整文字粗細（例如，細、常規、粗）。
*   **語言**：設定應用程式語言。您可以選擇特定語言或讓其跟隨系統預設設定。

### 行為
*   **網路超時**：設定應用等待網路請求回應的時間，從 1 到 60 秒。
*   **代理設定**：為網路請求配置代理伺服器。
*   **快取設定**：管理應用程式的快取，包括清除快取資料以釋放空間。
*   **GIF 動畫**：控制動畫 GIF 的播放方式：**關閉**（靜態影像）、**一次**（播放一次）或 **循環**（連續播放）。

### 影片和音訊
*   **影片自動播放**：一個開關，用於控制影片在出現在螢幕上時是否自動開始播放。
*   **影片背景播放**：啟用此功能可在導航離開或切換到其他應用後繼續收聽影片中的音訊。
*   **與其他混合**：允許 Synura 的音訊與其他應用的音訊同時播放。
*   **直播回放 DVR 小時數**：對於直播流，選擇保留多少小時的廣播可供向後搜尋（從 0 到 6 小時）。

### 隱私和安全
*   **管理設定**：配置各種隱私和安全選項以控制儲存和分享哪些資料。

### 關於
*   **開源許可證**：檢視幫助 Synura 執行的開源軟體的許可證。
