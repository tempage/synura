# 歡迎使用 Synura！

## 什麼是 Synura？
Synura 是一款多功能應用程式，允許您使用稱為「擴充功能（Extensions）」的強大迷你應用程式瀏覽來自各種來源的內容。可以把它想像成一個瀏覽器，但您使用的是擴充功能而不是網站，以乾淨的原生應用格式獲取內容。

## 使用者核心概念

*   **擴充功能發現**：輸入網域名稱（例如 `example.com` 或 `https://example.com`）以自動從該網域獲取 `synura.js` 檔案。如果未提供通訊協定，則預設使用 `https://`。這是從官方網站安裝擴充功能的主要方式。
*   **直接安裝**：輸入完整的 URL（例如 `https://raw.githubusercontent.com/user/repo/main/synura.js`）以安裝特定的擴充功能指令碼。**安全提示**：此方法僅限於受信任的網域（如 GitHub、GitLab 等），以防止惡意程式碼執行。請勿對一般網域使用此方法。
*   **白名單驗證**：為了安全起見，直接 URL 安裝將根據允許網域的白名單進行驗證。網域發現會繞過此檢查以允許探索。
*   **擴充功能**：這些是獲取和顯示內容的小型外掛程式。例如，您可以擁有新聞網站、影片平台或社群媒體動態的擴充功能。您可以安裝新擴充功能來擴展您可以使用 Synura 做的事情。
*   **執行環境（Runtimes）**：當您打開一個擴充功能時，它在一個「執行環境」中執行。您可以同時打開多個執行環境，就像在 Web 瀏覽器中有多個分頁一樣。每個執行環境都是擴充功能的一個單獨實例。您可以在它們之間切換，甚至可以為同一個擴充功能擁有多個執行環境。
*   **書籤**：發現了有趣的東西？您可以將當前視圖添加為書籤以供稍後儲存。書籤會儲存視圖的確切狀態，以便您隨時返回。

## 瀏覽應用程式

### 主畫面
應用程式的主畫面是您管理執行環境的地方。頂部欄（應用程式欄）是您的主要導航工具。

### 應用程式欄

應用程式欄有幾個圖示，可幫助您導航和管理內容。某些圖示具有透過長按存取的**隱藏捷徑**：

*   **`+` (新增)**：
    *   **輕觸**：打開一個新的執行環境。您可以選擇已安裝的擴充功能或輸入網站網域/URL 來安裝新的擴充功能。
    *   **長按**：打開**擴充功能管理**螢幕以查看有關已安裝擴充功能的詳細資訊。
*   **下拉選單（中間）**：顯示當前活動的執行環境。輕觸以在打開的執行環境之間切換，或在下拉選單上向左/向右滑動以循環瀏覽它們。
*   **`X` (關閉)**：關閉當前執行環境。
*   **`↻` (更新)**：*僅在開發者模式下可見。* 從來源更新當前擴充功能。
*   **`✨` (AI)**：
    *   **輕觸**：打開**AI 選單**以進行快速操作（摘要、翻譯等）。
    *   **長按**：打開**AI 設定**以配置提供者和偏好設定。
*   **`☆` (新增書籤)**：
    *   **輕觸**：將當前視圖儲存到您的書籤。
    *   **長按**：直接轉到您的**書籤**列表。
*   **`🔖` (書籤)**：查看已儲存的書籤列表。
*   **`⚙️` (設定)**：打開主設定螢幕。

如果螢幕太窄，某些選項可能會移動到三點選單中。

### AI 按鈕 (`✨`)
輕觸應用程式欄中的 **AI 按鈕**以打開 **AI 選單對話方塊**。這為您提供了針對當前視圖的按需 AI 驅動功能：

*   **摘要**：快速獲取 AI 生成的螢幕內容摘要。
*   **翻譯**：將內容翻譯成您的目標語言（在 AI 設定中配置）。
*   **自訂提示**：輸入您自己的指令讓 AI 分析內容。
*   **分享到外部 AI**：將當前視圖的內容匯出到裝置上的外部 AI 應用，如 ChatGPT 或 Gemini。
*   **快取切換**：控制是使用快取的 AI 結果還是強制進行新分析。

有關詳細的 AI 配置，請轉到 **設定 > AI 設定**，您可以在其中：
*   配置您首選的 AI 提供者（Gemini、OpenAI、DeepSeek、Claude）。
*   設定翻譯的來源語言和目標語言。
*   選擇分析搜尋範圍（深度僅在列表視圖中）和設定檔（摘要、解釋、簡化、事實查核、批評、見解）。
*   調整摘要長度偏好設定。
*   查看權杖使用統計資訊。
*   管理每個提供者的 API 金鑰。

### 書籤
書籤螢幕顯示您所有已儲存的視圖。

*   **視圖快照**：輕觸書籤會打開頁面在您儲存時的**快取快照**。這非常適合在無需網際網路連線的情況下快速參考資訊。
*   **還原視圖**：要再次與頁面互動（例如，點擊連結、重新整理資料），請尋找**還原圖示**。輕觸此圖示將重新連線到擴充功能，並在新的執行環境中還原視圖。

## 設定 (`⚙️`)

設定螢幕允許您微調 Synura 體驗的幾乎每個方面。

### 擴充功能
*   **安裝新擴充功能**：輕觸應用程式欄中的 **`+`** 按鈕並輸入網站網域（例如 `https://example.com`）。如果該網站支援 Synura，將自動發現並安裝擴充功能。
*   **管理擴充功能**：輕觸**管理**以查看已安裝擴充功能的列表，您可以在其中更新或刪除它們。

### 外觀
*   **調整內容密度**：使用滑桿使內容看起來更分散或更緊湊。您將看到它如何影響列表和卡片的即時預覽。
*   **色彩主題**：透過在**淺色**、**深色**和 **Monokai** 配色方案之間進行選擇來個性化應用程式的外觀。
*   **字型粗細**：根據您的喜好調整文字粗細（例如，細體、常規、粗體）。
*   **語言**：設定應用程式語言。您可以選擇特定語言或讓其跟隨系統預設設定。

### 行為
*   **網路逾時**：設定應用程式等待網路請求回應的時間，從 1 到 60 秒。
*   **Proxy 設定**：為網路請求配置 Proxy 伺服器。
*   **快取設定**：管理應用程式的快取，包括清除快取資料以釋放空間。
*   **GIF 動畫**：控制動畫 GIF 的播放方式：**關閉**（靜態影像）、**一次**（播放一次）或**循環**（連續播放）。

### 影片和音訊
*   **影片自動播放**：一個開關，用於控制影片在螢幕上出現時是否自動開始播放。
*   **影片背景播放**：啟用此功能可在導航離開或切換到另一個應用程式後繼續收聽影片中的音訊。
*   **與其他混合**：允許 Synura 的音訊與其他應用程式的音訊同時播放。
*   **直播回放 DVR 小時數**：對於直播串流，選擇保留多少小時的廣播可供向後搜尋（從 0 到 6 小時）。

### 隱私權與安全性
*   **管理設定**：配置各種隱私權和安全性選項以控制儲存和共用哪些資料。

### 關於
*   **開源授權**：查看幫助 Synura 執行的開源軟體的授權。
