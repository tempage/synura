# 欢迎使用 Synura！

## 什么是 Synura？
Synura 是一款多功能应用程序，允许您使用称为“扩展程序”（extensions）的强大迷你应用程序浏览来自各种来源的内容。可以把它想象成一个浏览器，但您使用的是扩展程序来获取干净、原生应用格式的内容，而不是网站。

## 用户核心概念

*   **扩展程序发现**：输入一个域名（例如 `example.com` 或 `https://example.com`）以自动从该域名获取 `synura.js` 文件。如果未提供协议，则默认使用 `https://`。这是从官方网站安装扩展程序的主要方式。
*   **直接安装**：输入完整的 URL（例如 `https://raw.githubusercontent.com/user/repo/main/synura.js`）以安装特定的扩展程序脚本。**安全提示**：此方法仅限于受信任的域名（如 GitHub、GitLab 等），以防止恶意代码执行。请勿对通用域名使用此方法。
*   **白名单验证**：为了安全起见，直接 URL 安装将根据允许域名的白名单进行验证。域名发现绕过此检查以允许探索。
*   **扩展程序**：这些是获取和显示内容的小型插件。例如，您可以拥有一个新闻网站、视频平台或社交媒体源的扩展程序。您可以安装新的扩展程序来扩展您可以使用 Synura 做的事情。
*   **运行时 (Runtimes)**：当您打开一个扩展程序时，它在一个“运行时”中运行。您可以同时打开多个运行时，就像在 Web 浏览器中有多个标签页一样。每个运行时都是扩展程序的一个单独实例。您可以在它们之间切换，甚至可以为同一个扩展程序拥有多个运行时。
*   **书签**：发现了有趣的东西？您可以将当前视图添加为书签以供稍后保存。书签保存视图的确切状态，因此您可以随时返回。

## 在应用中导航

### 主屏幕
应用的主屏幕是您管理运行时的地方。顶部栏（应用栏）是您的主要导航工具。

### 应用栏

应用栏有几个图标：

*   **`+` (添加)**：点击此按钮打开一个新的运行时。您可以选择已安装的扩展程序或输入网站域名来安装新的扩展程序。
*   **下拉菜单 (中间)**：显示当前活动的运行时。点击它可查看所有打开的运行时列表并在它们之间切换。您也可以在下拉菜单上向左或向右滑动以快速切换。
*   **`X` (关闭)**：关闭当前运行时。
*   **`☆` (添加书签)**：点击此按钮将当前视图保存到您的书签。
*   **`🔖` (书签)**：带您进入已保存的书签列表。
*   **`⚙️` (设置)**：打开设置屏幕，您可以在其中自定义 Synura。

如果屏幕太窄，这些选项将折叠到右侧的三点菜单中。

### 书签
书签屏幕显示您所有保存的视图。

*   **视图快照**：点击书签会打开页面在您保存时的**缓存快照**。这非常适合在无需互联网连接的情况下快速参考信息。
*   **恢复视图**：要再次与页面交互（例如，点击链接、刷新数据），请查找**恢复图标**。点击此图标将重新连接到扩展程序并在新的运行时中使视图恢复活动。

## 设置 (`⚙️`)

设置屏幕允许您微调 Synura 体验的几乎每个方面。

### 扩展程序
*   **安装新扩展程序**：点击应用栏中的 **`+`** 按钮并输入网站域名（例如 `https://example.com`）。如果网站支持 Synura，将自动发现并安装扩展程序。
*   **管理扩展程序**：点击 **管理** 查看已安装的扩展程序列表，您可以在其中更新或删除它们。

### 外观
*   **调整内容密度**：使用滑块使内容看起来更分散或更紧凑。您将看到它如何影响列表和卡片的实时预览。
*   **颜色主题**：通过在 **浅色**、**深色** 和 **Monokai** 配色方案之间进行选择来个性化应用的外观。
*   **字体粗细**：根据您的喜好调整文本粗细（例如，细、常规、粗）。
*   **语言**：设置应用程序语言。您可以选择特定语言或让其跟随系统默认设置。

### 行为
*   **网络超时**：设置应用等待网络请求响应的时间，从 1 到 60 秒。
*   **代理设置**：为网络请求配置代理服务器。
*   **缓存设置**：管理应用程序的缓存，包括清除缓存数据以释放空间。
*   **GIF 动画**：控制动画 GIF 的播放方式：**关闭**（静态图像）、**一次**（播放一次）或 **循环**（连续播放）。

### 视频和音频
*   **视频自动播放**：一个开关，用于控制视频在出现在屏幕上时是否自动开始播放。
*   **视频后台播放**：启用此功能可在导航离开或切换到其他应用后继续收听视频中的音频。
*   **与其他混合**：允许 Synura 的音频与其他应用的音频同时播放。
*   **直播回放 DVR 小时数**：对于直播流，选择保留多少小时的广播可供向后搜索（从 0 到 6 小时）。

### 隐私和安全
*   **管理设置**：配置各种隐私和安全选项以控制存储和共享哪些数据。

### 关于
*   **开源许可证**：查看帮助 Synura 运行的开源软件的许可证。
